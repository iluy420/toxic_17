Прерывания:

выбор функции прерывания осуществляется с помощью занесения номера службы в регистр AH
mov AH, 4Ch
int 21h - вызов функции DOS
	Для вывода одного символа на экран ПК используется функция 02 прерывания 21h:
		mov DL, <код выводимого символа>
		mov AH, 2
		int 21h
	Функция 9: Для вывода на экран строки (последовательности символов):
		DS:DX := начальный адрес строки
		mov AH, 9
		int 21h.
	Функция 4Ch: завершение программы
		mov AL, <код завершения>
		mov AH, 4Ch
		int 21h

int 10h - Прерывание 10h: службы ROM BIOS для видеодисплея
	Служба 0h используется для выбора одного из четырнадцати режимов выдачи, показанных в таблице. Режимы с 0h по 6h применимы для стандартного цветного графического адаптера (CGA); режим 7h применяется для монохромного режима (MDA); режимы с 0Dh по 10h были добавлены для улучшенного графического адаптера (EGA); наконец, режимы 11h - 13h были внедрены на многоцветном графическом массиве (MCGA) (PS/2 моделей 25 и 30) и видеографическом массиве (VGA) (PS/2 моделей 50, 60 и 80).
	
	Служба 06h (в десятичной нотации 6) и ее напарник служба 07h (в десятичной нотации 7) используются для задания на экране прямоугольной области текстового окна и прокручивания его содержимого вверх или вниз на одну или более строк. 
		AH = 06h AL = число строк, подлежащих прокрутке (прокрутка CH = номер строки для верхнего лев. угла вверх)
		AH = 07h CL = номер колонки для верх. левого угла (прокрутка DH = номер строки для нижнего прав. угла вниз)
		DL = номер колонки для ниж. правого угла
		BH = атрибут для выдачи пустых строк
	
	Служба 02h (в десятичной нотации 2) устанавливает позицию курсора, используя координаты строки и колонки экрана. Значения регистров при установке положения курсора с помощью службы 02h 
		Номер службы Параметры
		AH = 02h DH = номер строки
		DL = номер колонки
		BH = номер страницы
	
int 16h - Работа с клавиатурой
	AH = 00h - читать следующий символ из буфера
	AH = 01h - определить наличие символов в буфере
	AH = 02h - определить статус клавиш-модификаторов (Shift/Control/Alt)
		mov ah, 00h
		int 16h ; после этого в регистре al окажется ASCII-код символа, а в регистре ah - сканкод.
Если символа в очереди не оказалось, то функция будет ждать символа. Если в al поместился 0, то в регистр ah помещается расширенный сканкод. Эта функция очень старая. Использовать эту функцию для современных клавиатур не рекомендуется.
	Функция 01 INT 16h служит для определения наличия символов в буфере клавиатуры.
		mov ah, 01h
		int 16h
		; ZF = 1 если символа нет
		; ZF = 0 если символ есть
		; AL - ASCII-код
		; AH - сканкод
Функция 02 INT 16h установка флагов соответствующие состоянию так называемых shift-клавиш. 
		mov ah, 02h
		int 16h ; AL = битовое поле
Описание бит регистра AL после вызова функции 02h:
		Бит	Клавиша
		0	Правый Shift
		1	Левый Shift
		2	Control
		3	Alt
		4	ScrollLock
		5	NumLock
		6	CapsLock
		7	Режим Insert
